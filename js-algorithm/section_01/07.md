### 내 풀이

```javascript
function solution(day, arr) {
  let answer = 0;

  for (let x of arr) {
    const carNumber = x.toString();
    if (parseInt(carNumber[1]) === day) {
      answer++;
    }
  }

  return answer;
}
```

숫자를 문자열로 바꿔서 day와 비교할 때는 다시 `parseInt` 함수를 쓰는 등 깔끔하지 못하다.

### 모범 답안

```javascript
function solution(day, arr) {
  let answer = 0;

  for (let x of arr) {
    if (x % 10 === day) answer++;
  }

  return answer;
}
```

10으로 나눈 나머지가 일의 자리 숫자인 것을 기억하자!
